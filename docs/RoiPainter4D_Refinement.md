## Refinement in RoiPainter4D

RoiPainter4Dでは，以下のRefinementツールを用いて領域を編集可能です．

<img align="left" width="300" src="assets/mode_switch_menu.png" alt="mode switch menu">
<strong> Refinement </strong> <br>
<a href="#r1_ref_stroketrim"> S-1) mode switch > Ref Stroke Trim </a> <br>
<a href="#s2_ref_curvedeform"> S-2) mode switch > Ref Curve Defrom </a> <br>
<br clear="left"/>

<hr>

<a id="r1_ref_stroketrim"></a>

### Ref Stroke Trim

トリミングした領域を別領域にすることができます．


<a id="r2_ref_curvedeform"></a>

### Ref Curve Deform

正確な境界を曲線で直接指定することにより，領域を編集できます．
