# データの出力・保存
RoiPainterで編集・作成したデータは，目的に応じて様々な形式でエクスポートすることができます．

## 1. マスクデータの保存
マスク画像を保存します．

| 形式 | 拡張子 | 特徴・用途 |
| :--- | :--- | :--- |
| **4D Mask** | `.msk4` | 独自の4D対応形式です．マスク名・色・ロック状態等を保持します．([仕様はこちら]()) |
| **MetaImage** | `.mha` | 医療画像処理で一般的な形式です．ITK-SNAPや3D Slicer等の他ソフトと互換性があります． |
| **Raw 8bit** | `.traw3d_ub` | 8bitバイナリ形式です． |
